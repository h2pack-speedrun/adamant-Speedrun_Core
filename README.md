# adamant-ModpackSpeedrunCore

Coordinator for the Speedrun modpack. Owns the pack identity, config, and default profiles. All orchestration (discovery, UI, hashing, HUD) is provided by `adamant-Modpack_Framework`.

Part of the [Speedrun](https://github.com/h2pack-speedrun/speedrun-modpack).

## Install

Install via r2modman or Thunderstore. Requires `adamant-Modpack_Lib` and `adamant-Modpack_Framework`.
